ローカルファースト

予定はブラウザの手元(IndexedDB)に複製。サーバーは正本を持たず、オフラインでも開ける。

GitHub 連携

issue・PR を作業キューに。予定と実績を突き合わせ、かけた時間を可視化する。

MCP サーバー

Claude などのエージェントが予定を読み書きできる公式 API。空き時間の提案も。 接続ガイド →

デスクトップ版

Tauri 製アプリを Homebrew で。トレイ常駐・グローバルショートカット・ウィンドウ記憶。

不在・参加ステータス

不在(OOO)は端のレールに、未返信・仮・不参加を見た目で区別して表示。

カレンダーブロック

別カレンダーへ予定を自動ミラー。多重予約を防ぐ。

オープンソース、セルフホスト可能。

MIT ライセンスのオープンソース。自分の Cloudflare アカウントにデプロイして、自前で運用することもできる。
