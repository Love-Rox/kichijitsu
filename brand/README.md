# kichijitsu ブランドアセット

コンセプト: **枡の週**。7つの枡（カレンダーのセル）＝1週間。日々は薄墨で平等に並び、
選ばれた日だけ朱の枡が**手で押印されたように少し傾いて (-8°)** 載る。
日々は平等で、選ぶ行為が吉日を作る——「思い立ったが吉日」。

## アセット

| ファイル | 用途 |
|---------|------|
| `mark-week.svg` | 横長マーク。ヘッダー・README・OG 画像の素材 |
| `tile.svg` | アプリアイコン (48px〜)。月グリッド 3x2 版 |
| `favicon.svg` | ファビコン。16px でも読める 2x2 簡約版 |

ワードマークは SVG を持たず、アプリ内で文字組み + j の点（tittle）を傾いた朱の枡に
置き換えて描画する（`apps/web/src/components/Logo.tsx`）。

## 枡オーナメント（UI 全体への展開）

枡ひとつ＝「一日」「一項目」を最小単位の記号として UI 全体で使う。

- **薄墨の正立した枡** = ふつうの項目（箇条書きの点、メニュー項目、未選択）
- **枠だけの空枡** = 未完了（チェックボックスの unchecked）
- **朱で -8° 傾いた枡** = 選ばれた・済んだ・今日（メニュー現在地、完了チェック、今日の日付バッジ）
- 傾きと朱は必ずセット。「人の手が判を押した」の意味。薄墨の枡は常に正立
- サイズは文字に対して 0.48〜0.52em
- 朱の枡は1つのまとまりにつき基本1個（完了リスト等の例外時は他の朱要素を置かない）
- アニメーションは「置く・押す」の2語彙のみ（インジケーターと共通）。回転・バウンド追加禁止
- 適用先の例: 箇条書き・設定メニュー・チェックボックス（完了=押印）・週ヘッダーの今日バッジ

## ルール

- ワードマークの j の点（tittle）は**常に朱の枡**（ユーザー決定 2026-07-19。
  マーク併用時も朱のまま——ロゴ領域内はこの2つで1つのアクセントとみなす）
- j の枡は i の点と**高さ中央揃え**にする（i の tittle の中心と枡の中心を一致させる）
- ロゴ領域の外では、アクセントの朱はひとまとまりにつき1つの原則を守る
- 朱の枡の傾きは常に **-8°**。傾きが変わると別人になる
- モーション＝**枡インジケーター**（同期中・展開中などのローディング表示。ロゴがそのまま動く）:
  - 1周 1820ms、枡の登場ずれ 130ms、イージング cubic-bezier(.2,.7,.3,1)
  - 通常の枡は scale .55→1 で静かに置かれる。朱の枡のみ押印オーバーシュート
    (scale 1.45→1→1.06→1、常に rotate(-8deg))
  - `prefers-reduced-motion: reduce` では静止した完成形マークを表示
  - 停止時は現在の周回を完走してからフェードアウト（ぶつ切り禁止）
  - 実装は `apps/web/src/components/MasuIndicator.tsx`
